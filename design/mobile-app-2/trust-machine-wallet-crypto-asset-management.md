# Trust Machine Wallet - Crypto Asset Management

## 1. Project Overview

Trust Machines Wallet is a mobile application designed for cryptocurrency asset management, focusing on optimizing the user experience within a rapidly growing digital finance market.

* **Role**: UI/UX Designer.
* **Device**: Mobile App.
* **Project type**: Design based on the specs and wireframe.
* **Goal**: To design a modern, secure, and easy-to-use crypto wallet application.

## 2. Challenges

The project faced three primary challenges to establish a market presence:

* **Saturated** **Market**: Developing a concept that could stand out among numerous existing crypto wallet applications.
* **The Balance**: Ensuring the design was unique and fresh while remaining familiar enough that users would not feel overwhelmed.
* **Aesthetics & Functionality**: Implementing a dark theme that was both visually appealing and highly functional.

## 3. Solution

To address these challenges, the design process focused on:

* **Competitor Research**: Analyzing existing crypto wallets (such as Phantom, Hiro, and Exodus) to understand common design patterns and user expectations.
* **Fresh Visual Style**: Introducing a sleek dark theme combined with modern design elements.
* **Result**: Created an attractive, user-friendly interface that stands out without sacrificing usability.

## 4. Key Features

The app integrates standard and advanced features for a comprehensive user experience:

* **Secure** **Authentication**: A sign-up process with memorization screens for new private keys, passphrase login, and FaceID support to bypass manual passwords.
* **Transaction** **Management**: Support for sending and receiving BTC, STX, or other tokens via a QR scanner, with detailed transaction history and status screens.
* **Diverse Asset Support**: Management of token balances (including SIP-010 standards) and a dedicated "Collectibles" screen for a vibrant NFT display.
* **Advanced Functionality**: Token swap capabilities using Magic Bridge and a built-in DApp browser (wallet injector).

## 5. Design System

The project was built upon a strict Style Guide to ensure consistency:

* **Typography & Colors**: Utilization of a deep dark color palette paired with highlight colors for different states like success, error, and warnings.
* **Components**: Meticulously designed UI components including Buttons, Input Fields, Checkboxes, and Toggles to optimize user interaction.

## 6. Takeaways

This project marked significant growth in professional skills:

* **Execution Skills**: Greatly improved the ability to translate wireframes into a polished, high-fidelity UI.
* **Product Thinking**: Gained deeper insight into designing for financial applications, where building trust and ensuring practicality are top priorities.
