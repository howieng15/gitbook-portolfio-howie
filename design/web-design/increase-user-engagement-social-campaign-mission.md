# Increase User Engagement - Social Campaign Mission

**Type:** Web Design, UI/UX, Brand & Visual Design\
**Website**: Closed\
**Project:** A social campaign platform that boosts user participation through missions\
**Role:** Web & Visual Designer\
**Year:** 2024

## **Overview**

I designed a social campaign mission interface to help users engage more.\
The goal was to make the platform fun and easy to use.\
Users earn points by completing tasks and interacting with features.

## **Scope of Work**

* Created a campaign website.
* Designed graphics for promotional materials.
* Worked with the marketing and development teams to ensure smooth execution.

## **Challenges**

* The design had to be simple but visually appealing to make users want to join missions.
* I needed to create a points system that was clear and guided users well.

## **Solution**

* I chose a universe theme to give a futuristic and unified visual style.
* I made a clean, single‑page layout that works well on mobile and desktop.
* I used bold colors and strong typography to highlight main actions.
* I added clear buttons and simple navigation to help users flow easily through the interface.

## **My Contributions**

* Designed the visual identity with a universe and space concept.
* Created the points system layout and mission flow.
* Built a responsive one-page layout for both mobile and desktop.
* Focused on visual cues and CTA buttons to improve user action.

## **Design Focus**

* Keep interface neat and intuitive.
* Make actions clear through visuals and layout.
* Design for a fast and engaging user experience.
* Ensure visual style is consistent with brand theme.

## Outcomes

* The campaign successfully increased the follower base on 3 platforms:
